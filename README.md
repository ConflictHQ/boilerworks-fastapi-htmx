# Boilerworks FastAPI + HTMX

> Lightweight async Python with server-rendered HTMX and Alpine.js.

**Status:** Planned

FastAPI paired with HTMX for teams that want Python's async performance with minimal frontend complexity. Ideal for internal tools, dashboards, and lightweight CRUD apps where a full JS framework is overkill.
